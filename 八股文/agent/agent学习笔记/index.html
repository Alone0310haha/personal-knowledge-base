    Agent学习笔记

    语言模型是自然语言的核心，它的任务是计算下一个词/句子出现的频率。 在深度学习兴起之前，主流思想认为，一个句子出现的概率等于其中每个词出现的概率的乘积。但直接计算这个概率不太可能。所以引入了马尔可夫假设，它的核心思想是只用现在来预测未来。 比如说，我现在要从北京去上海，是坐飞机还是坐火车还是坐公交，都和我怎么到北京的无关。

    N-gram模型是用马尔可夫假设和最大似然估计构建出的一种语言预测模型。 这个模型就是用来从你前面说的n-1个词来预测第n个词是什么？它根据前面这n-1个词去一个巨大的词典里面找历史n-1个词后的第n个词出现次数最多的词，模型就预测你会说这个。

    假设你正在打字，刚打完“我 爱 吃”，现在想知道下一个词是什么。

    假设是 3-gram 模型，它现在眼里只有“我 爱 吃”这三个词。 模型会去翻阅一本巨大的“人类语料库“，看看在“我 爱 吃”这三个词后面，历史上最常接的是什么词？如果历史上“我 爱 吃”后面接“苹果”出现了 100 次，接“火锅”出现了 50 次，接“石头”出现了 0 次。模型就会反推：既然“苹果”出现的概率（似然）最大，那么下一个词**最合理的猜测**就是“苹果”。

    它的局限性在于它没有长期记忆。

    在此基础上研究出了**前馈神经网络语言模型**。该模型将词抽象成向量空间中的一个点，语义相近的词在空间中对应的位置也相近。本质上就是一个 f(x)=y的函数，最开始肯定是瞎猜，预训练其实就是在调整这个f(x)=y，后续还可以通过反向传播和梯度下降来调整。

    余弦相似度就是用来知道两个词意思相近的。

    loop engineering其实就是将前一个输出当作后一个输入，一直循环。

    人工智能的本质就是尽可能的将问题函数化，最开始的时候追求一定要找到一个能完全描述问题的函数，后来发现做不到，就找出一个能近似描述的函数。 最开始的时候是从线性函数（y=ax+b）开始描述的，后来发现线性函数不能满足需要了，就在外面套了一层非线性变换（平方，e，sin等），这个非线性变换就是激活函数。 所以就从y=ax+b演变成了 f(x) = g(ax+b)，套一层不够就继续加叠加。 但我们的输入和输出都可以有多个，中间层也可以有多个，就这样一层套一层，一层套一层，最后形成的网状结构就是神经网络。 ![[Pasted image 20260629210239.png]]

    第一讲的核心思想是讲神经网络的思想，为什么会出现神经网络这个东西 =》人们想把现实世界的问题抽象成函数。

    第二讲是讲怎么评价这个函数算是真实的模拟了现实世界的问题。=》损失函数

    首先要有一定的基础数据，以最简单的方式描述的话就是坐标系中的一堆点。现在有一个线性变换，评价这个线性变换的拟合程度就是判断所有点到这根线的距离。把所有点的距离加起来就是误差，误差越小就越拟合。计算这个误差的函数就叫做损失函数。 ![[Pasted image 20260629211648.png]]

    注：此处计算误差的时候用的是均方误差，即先平方再平均，这样能消除极值和样本数量的影响，标准比较统一。

    通过寻找一个线性函数来拟合x和y的关系的过程叫线性回归。其实这个寻找过程是一点一点试出来的，每次找一组(a,b)计算一次损失函数，每次都把参数朝着让损失函数变小的方向调，调整的速率叫学习率。 调整的方向也可以用一个函数来表示，这个就是梯度。根据梯度和学习率最终找到让损失函数最小的（a,b）的过程叫梯度下降。

    模型先顺着算一遍结果（正向传播），然后算算错得有多离谱（损失函数），接着拿着这个误差，利用**链式法则**一层层往回倒推（反向传播），算出每个旋钮该怎么调（梯度），最后乘以**学习率**（步子大小），把旋钮调好。这样就是一次训练。

    神经网络并不是越大越好，在训练的过程中需要避免函数的过度拟合。

    人类为了处理自然语言的一系列问题： 首先是将词转换成计算机能够识别的数字（即编码），通过编码词形成的向量叫词向量。在编码词向量的过程中有一种方式叫one-hot，它的思想是词表有10w个词，就用一个10w的向量表示，其中只有一个1，剩下的都为0。这种方式维度太高。所以衍生出了词嵌入（embedding)。词嵌入所需要得到的矩阵叫嵌入矩阵。词向量所在的高维空间叫词空间。词向量之间的关系可以用点积或者余弦相似度来计算。有词向量之后就能输入到神经网络进行计算。经典的神经网络无法表示词的先后顺序，所以增加了一个隐藏状态在词和词之间传递。这个结构叫循环神经网络RNN。

    RNN存在的问题有：1. 无法捕捉长期依赖 2.由于隐藏状态需要根据前置维度算出来，所以不能并行计算。
