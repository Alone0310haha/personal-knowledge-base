# 大模型训练原理：梯度下降，从一条直线讲起

## 来源

- [大模型训练原理：梯度下降，从一条直线讲起](https://www.bilibili.com/video/BV1mV2tBgE3G/)

## 讨论视角

- 以“最简单的一层线性模型（拟合直线）”做最小可解释案例，把训练拆成可计算的步骤
- 用“损失函数的曲线形状”解释：训练不是靠猜最优解，而是靠局部斜率（梯度）决定下一步往哪走、走多远
- 把工程里最常见的两个超参数（学习率、batch size）放在同一个框架里解释其意义与坑点

## 核心论点与分论点

### 核心论点

- 训练的本质是：定义损失函数衡量预测与真实的差距；对参数求梯度；沿着使损失下降的方向、按学习率迈小步反复更新参数，这个过程叫梯度下降。

### 分论点 1：从最小模型建立“训练”这件事的全流程

- 模型：单个线性层，输入/输出都是标量
  - \(y_{out}=Wx+b\)
  - \(W,b\) 是需要训练得到的参数
- 数据：大量 \((x,y)\) 样本点，整体分布在某条“真实直线”附近（带噪声），训练目标是找出一条尽量贴近数据规律的直线

### 分论点 2：损失函数把“差多少”变成可优化的数

- 训练需要一个标量指标衡量预测与真实的差距：Loss
- 绝对值误差在 0 点不可导（作者用“工程上麻烦”解释），因此常用平方误差：
  - 单样本：\(\mathrm{Loss}=(y_{out}-y)^2=(Wx+b-y)^2\)
  - 越接近真实 \(y\)，Loss 越小

### 分论点 3：梯度是“局部方向指示器”，学习率决定“步长”

- 不需要直接求全局最优点；只需要知道当前点的变化趋势
- 对当前要更新的参数（如 \(W\) 或 \(b\)）求导得到梯度：
  - 梯度为负：增加该参数会让 Loss 变小
  - 梯度为正：减少该参数会让 Loss 变小
  - 梯度绝对值越大：离最优点越远（局部变化更陡）
- 更新规则（示意）：
  - \(W \leftarrow W - \eta \frac{\partial \mathrm{Loss}}{\partial W}\)
  - \(b \leftarrow b - \eta \frac{\partial \mathrm{Loss}}{\partial b}\)
  - \(\eta\) 即学习率（Learning Rate），属于超参数

### 分论点 4：单样本会被噪声拉扯；batch 平均能“收敛到规律”

- 真实数据带噪：即使参数已经接近真实规律，某个偏离较大的样本也可能把更新方向“拉偏”
- 解决：一次使用一批样本（batch）计算平均损失再求梯度更新
  - 平均平方误差：MSE（Mean Squared Error）
  - batch size 越大，梯度估计越稳定；越小，更新更“抖”但更频繁（作者的重点在于“用平均对抗噪声”）

## 关键数据 / 案例 / 结论

- 例子模型：\(y_{out}=Wx+b\)
- 单样本平方误差：\((Wx+b-y)^2\)
- 多样本损失：对一个 batch 内的平方误差求和再除以样本数（MSE）
- 参数 vs 超参数：
  - 参数：\(W,b\)（由数据训练得到）
  - 超参数：学习率 \(\eta\)、batch size（由人设定）
- 两个新手易踩坑（作者强调的“请分清”）：
  - 梯度下降是对损失函数做下降，不是对模型函数本身
  - 求梯度时，“变量”是当前要更新的参数（\(W\) 或 \(b\)），训练数据 \(x,y\) 在该步视为常量
- 训练结果示例（作者演示的一次训练曲线结果）：最终拟合到接近 \(y\approx 1.5x+2.5\) 的直线（文字稿中示例输出约为 \(y\approx 1.4994x+2.4845\)）

## 思维导图（Markmap）

```markmap
# 梯度下降训练原理（从一条直线讲起）
## 最小模型
### y_out = W x + b
### 参数：W、b
### 数据：(x,y) 点云围绕真实直线（含噪声）
## 损失函数（衡量差距）
### 单样本：Loss = (y_out - y)^2
### 目的：让 Loss 变小
### 绝对值误差在 0 点不可导 -> 工程上常用平方误差
## 梯度（方向信息）
### 求导：dLoss/dW、dLoss/db
### 符号决定方向
#### 梯度 < 0 -> 增大参数让 Loss 下降
#### 梯度 > 0 -> 减小参数让 Loss 下降
### 绝对值决定“远近/陡峭程度”
## 参数更新（梯度下降）
### W <- W - lr * dLoss/dW
### b <- b - lr * dLoss/db
### 学习率 lr：步长（超参数）
#### 太大：易越过最低点/不稳定
#### 太小：收敛慢
## Batch 与 MSE
### 单样本更新易被噪声拉扯
### Batch size：一次用多少样本（超参数）
### MSE：对 batch 内 loss 求平均
### batch 足够大 -> 平均效果更接近真实规律
## 易混点
### 下降的是“损失函数”，不是“模型函数”
### 求导变量是参数 W/b，训练数据 x/y 在该步视为常量
```
