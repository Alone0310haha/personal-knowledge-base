**分析风格说明**：

#### 4.1 信息提炼型

做这种分析的目的是把视频的干货榨干。

你必须分析以下维度：
- 视频是从什么角度出发进行的讨论
- 视频的核心论点和分论点
- 列出视频中提到的关键数据、案例和结论
- 以markmap代码的格式画一个视频的思维导图大纲

#### 4.2 速览概览型

做这种分析的目的是想知道讲了什么故事或者发生了什么。

你必须分析以下维度：
- 视频讲了什么故事或者发生了什么
- 事件发生的时间线，剧情梗概，里程牌，重大事件，关键转折


#### 4.3 行动指南型

做这种分析的目的是能跟着视频做。

你必须分析以下维度：
- 请将视频中的操作方法拆解成一步一步的清单。
- 请总结教程里需要用到的工具、材料和注意事项。
- 请提取视频中提到的常见错误和解决方案。

#### 4.4 拆解金句型

做这种分析的目的是总结视频中能打动人的文字，或者能直接引用的素材。

你必须分析以下维度：
- 请摘录视频中最精彩、最有启发性的几句话。
- 请总结主讲人对某个话题的核心态度和价值观。
- 如果视频内容为辩论，请总结辩论的核心分歧点。

#### 4.5 立场分析型

做这种分析的目的是想客观了解视频讲了什么，以及它背后可能存在的倾向。

你必须分析以下维度：
- 请客观分析视频中正反双方各自的逻辑和论据。
- 请指出这个视频作者为了论证观点，使用了哪些主要论据？
- 请分析这个视频内容的立场和潜在倾向。

#### 4.6 质量分析型

做这种分析的目的是想知道视频的质量，以及后续应该从什么角度优化。

你必须分析以下维度：
- 开头是否足够吸引人？是否有明确的钩子（Hook）留住观众？
- 内容推进是否拖沓？高潮部分是否铺垫到位？
- 干货含量是否足够？是否存在无效废话？
- 在哪些节点适合插入引导点赞、评论或弹幕的内容？
- 针对上述问题，给出具体的修改方案或重写建议。
