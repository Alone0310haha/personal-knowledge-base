    The CORRECT way to use ChatGPT

    The CORRECT way to use ChatGPT

    讨论视角（作者从什么角度出发进行讨论）¶

    作者以“功能更新太快、概念营销太多”的用户视角切入，目标不是讲术语，而是给出一套可直接执行的使用准则：在真实任务中如何选择模型、何时用搜索/深度研究/画布（Canvas）等功能，让 ChatGPT 产出更稳定、返工更少、速度更快。

    核心论点与分论点¶

    核心论点¶

    正确使用 ChatGPT 的关键不是记住所有新功能，而是根据“任务风险与复杂度”选择合适的模型和工作模式，并在输入结构（分隔符/格式）与工作流（检索→分析→编辑迭代）上做工程化设计。

    分论点 1：先选对模型：低风险用 chat，高复杂/高风险用 reasoning¶

    • Chat（基础对话模型，视频用“GPT-”类作示意）：适合低风险、要快的任务。
    • Reasoning（推理模型，视频用“O number”类作示意）：适合重要或复杂任务，你愿意多等一点换更好的答案。
    • 作者的经验规则：默认选“最新的推理模型”，除非你明确知道任务很轻、追求极致速度。

    分论点 2：推理模型的 3 个提示词技巧¶

    1. 用分隔符把“任务指令”和“待分析材料”分开（例如 Task / Document 两段）。
    2. 不要写 “think step by step”
    3. 视频观点：这句话可能帮助 chat 模型，但会伤害推理模型；推理模型本来就会分步思考。
    4. 示例不是必需（zero-shot 通常就够）
    5. 只有当输出不对时，再加 one-shot/few-shot 示例纠偏。

    分论点 3：搜索功能：Google vs ChatGPT Web Search 的分工¶

    • 单一事实：优先用 Google（更快、也更让人安心）。
    • “事实 + 简短解释/上下文”：用 ChatGPT 搜索（能把结果组织成可理解的叙述，支持追问）。
    • 两条使用建议：
    • 用于快速 fact check（作者称“还不错”）。
    • 在对话框用快捷命令快速开启搜索（视频类比 Notion 的 / 命令）。

    分论点 4：Deep Research：把“开很多标签页的研究”外包给 Agent¶

    • 定义：一个会花 10–20 分钟阅读大量链接并产出长报告的研究代理。
    • 适用：跨多来源的信息对比、行业/竞品研究、个人金融对比等。
    • 输入越详细越好；但可通过模板化方式降低写提示词成本（先用模板生成器/项目资料生成研究提示）。
    • 进一步增强：连接你有权限的数据源（例如 Google Drive），把内部资料与外部资料一起引用与对比。

    分论点 5：Canvas：需要反复编辑的任务，用“可版本化的编辑面板”更高效¶

    • 规则：如果你预计会对同一份输出改很多次，启用 Canvas。
    • 适用：写作/汇报材料/结构化文档，也适合 coding（作者主要用于文案写作）。
    • 三个技巧：
    • 用前进/后退在版本间跳转。
    • 用内置快捷操作对全文或选中片段做改写/建议。
    • 支持导出 Markdown，方便落盘到文档系统（例如再导入到 Google Docs）。

    关键案例与结论（视频中的例子）¶

    • 选模型的例子：
    • “哪些水果纤维最多？”（低风险）→ chat 即可。
    • “做一个含 ≥15g 纤维且 ≥20g 蛋白的素食早餐”（约束多）→ reasoning 更合适。
    • “成功从来不是最终”（模糊引用）→ chat 找出处即可；但“解释丘吉尔为何战后仍落选”（历史语境复杂）→ reasoning。
    • 写普通邮件（明确要点）→ chat；处理 20 条来回的冲突邮件并写“外交式回复”（高风险）→ 上传材料 + reasoning。
    • 搜索的例子：
    • NVDA 股价（单一事实）→ Google 快；但“最新财报电话会时间、股价涨跌、原因”→ ChatGPT 搜索更有上下文。
    • 苏黎世天气（单一事实）→ Google；“12/1–12/7 去苏黎世带什么衣服”→ ChatGPT 搜索更像顾问回答。
    • 疫苗接种率：单点查询 Google；“过去 5 年全球接种率并要表格”→ ChatGPT 搜索更方便。
    • Deep Research 的例子：
    • 对比 Nvidia/AMD/Intel AI 芯片路线图（基于最新财报与电话会）。
    • 对比美国高收益储蓄账户：费率、隐藏费用、按每月储蓄额做收益投影。
    • 企业用例：引用内部数据 + 外部行业报告做季度竞品对比报告。
    • Canvas 的例子：
    • 写绩效自评：上传公司 rubric → 生成大纲 → 反复删改、局部改写 → 生成高管摘要。
    • 文本模型常用“命令词”：
    • elaborate（扩写加细节）
    • critique（提前挑问题、做反驳压力测试）
    • rewrite（按语气/风格重写）
