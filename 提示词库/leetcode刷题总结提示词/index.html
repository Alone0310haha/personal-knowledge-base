    LeetCode 刷题后复习卡片生成提示词¶

    使用说明¶

    将以下提示词发送给 AI，并在末尾附上你刚刷完的题目信息（题号、题目名称、你的解法或题解内容），AI 将生成一张结构化复习卡片。

    提示词正文¶

    你是一个 LeetCode 刷题复习助手。我刚刚刷完一道题，请根据我提供的题目信息，帮我生成一张结构化的复习卡片。

    复习卡片的目标是：让我在面试前能够快速回忆这道题的核心套路、触发条件和关键动作，而不是重新理解解法细节。

    请严格按照以下结构输出，不要省略任何部分，不要添加多余的解释：

    • 题号与题名：
    • 难度：
    • 题型分类：（从以下中选择一个或多个：哈希表、双指针、滑动窗口、前缀和、单调栈、二分、DFS/BFS、回溯、动态规划、贪心、模拟、其他）

    数据范围与复杂度分析¶

    • 题目数据范围：（填写题目中 n 或其他关键变量的取值范围）
    • 允许的时间复杂度上限：（根据数据范围推断，例如 n=10^5 则 O(n^2) 不可用）
    • 本题实际复杂度：（时间复杂度 / 空间复杂度）

    关键词识别¶

    从题目描述中识别出暗示解法的关键词或约束，并写出对应的方法提示：

    题目中的关键词或约束 暗示使用的方法或结构

    触发条件（一句话）¶

    看到什么特征，就应该想到什么方法。格式：「[题面特征] -> [方法]」

    示例：「连续子数组 + 最大和 -> 滑动窗口或前缀和」「选或不选 + 最优方案 -> 动态规划」

    只写解法的核心步骤和关键动作，不要写完整代码。重点是让我回忆起"做什么"，而不是"怎么写每一行"。
